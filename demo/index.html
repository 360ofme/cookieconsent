Cookie Consent demo

Simple cross-browser cookie-consent plugin written in vanilla js.

If your browser is set in italian language, then the cookie-consent will be loaded with the newly defined it-lang (see app.js for the cookie-consent configuration), otherwise it will default to en-lang.


Go to github repo.

Check console window for more details about api.


  • CookieConsent.run({<config_object>});

  • CookieConsent.show({<optional_delay_value>});

  • CookieConsent.show_policy({<optional_delay_value>});

  • CookieConsent.hide({<optional_delay_value>});

  • CookieConsent.clearCookies();
