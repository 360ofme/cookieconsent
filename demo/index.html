CookieConsent demo

Simple, lightweight, responsive and extensible cookie consent! Github repo.

If your browser is set in italian language, then the cookie-consent will be loaded with the newly defined it-lang (see app.js), otherwise it will default to en-lang


  • CookieConsent.run({<config_object>});

  • CookieConsent.show({<optional_delay_value>});

  • CookieConsent.show_policy({<optional_delay_value>});

  • CookieConsent.hide({<optional_delay_value>});

  • CookieConsent.clearCookies();
